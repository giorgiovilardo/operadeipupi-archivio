# Opera dei Pupi — Archivio

Frontend for the Opera dei Pupi digital archive. Built with Svelte 5, TypeScript, DaisyUI, and Vite.

## Setup

```bash
npm install
cp .env.dist .env
```

Edit `.env` if you need to point to a different API instance.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Produces a static SPA in `dist/`, ready for deployment.
